# A11y Demo Site

Automated accessibility compliance and provenance demonstration.

## Key Features

- **Evidence Generation** — Scans HTML for accessibility violations
- **Cryptographic Provenance** — Signs evidence bundles to ensure tamper-proof results
- **Automated Advisories** — Converts findings into fix-oriented guidance
- **CI/CD Integration** — Block builds on regressions or warn on findings
- **Intentional Bugs** — Demonstrates real-world accessibility issues for learning

## Quick Start

```bash
npm install -g a11y-evidence-engine
pip install a11y-assist
./scripts/a11y.sh
```
